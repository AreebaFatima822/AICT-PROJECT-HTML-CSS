# AICT-PROJECT-HTML-CSS
" Project for AICT subject using HTML and CSS on GREEN TECHNOLOGY SHOWCASE "
Green Technology Showcase is a web-based project that highlights environmentally friendly technologies and sustainable innovations. The website provides information about renewable energy, green computing, and other eco-friendly solutions to promote sustainability awareness.

Technologies Used

HTML – for structuring the pages and content.

CSS – for styling the website, including layout, colors, and responsive design.

JavaScript – for adding interactivity, such as contact form validation.

Project Features

Home Page: Introduction with navigation and call-to-action buttons.

About Page: Background information and mission of the project.

Technologies Page: Cards and tables displaying green technologies and their benefits.

Contact Page: Form to submit user inquiries with validation.
